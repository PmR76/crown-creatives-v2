# Crown Creatives

A magical, creative, and supportive space designed to encourage artistry, resilience, and imagination.  
This project is built to be fully editable by admin, responsive across all devices, and centred around a day/night magical experience.

## Vision
Crown Creatives blends creativity with wellbeing, offering a calm, inspiring digital space that supports artistic expression and recovery.

## Features
- Transparent magical header and footer
- Day/Night mode with 8‑second transitions
- Intelligent gallery lanes
- Magical background engine (day + night)
- Editable pages, cards, and content
- Fully responsive layout (PC, tablet, mobile)

## Status
Actively in development — building the magical background engine, header, and hero system.
